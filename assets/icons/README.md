# Icon Files Needed

The extension requires PNG icons in the following sizes:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Create these icons and place them in this directory.
The icon should be a purple gradient with "IA" text (InterviewAce logo).

You can use the icon128.svg as a reference for creating PNG versions using any image editor.
